Required Dependencies

Before getting started, ensure you have the following dependencies installed:

  • Django 4.2.5
  • Pillow 10.0.0
  • python-dotenv 1.0.0

Getting Started

Follow these steps to set up and explore the project:

  1. Clone the Repository: Use Git to clone the project repository to your local machine.
  2. Install Dependencies: Run the following commands to install the required dependencies:
    3. 
pip install Django==4.2.5
pip install Pillow==10.0.0
pip install python-dotenv==1.0.0
  3. Create Database Tables: Initialize the database by running these commands:
    4. 
python manage.py makemigrations
python manage.py migrate
